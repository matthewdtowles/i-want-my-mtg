Lighthouse measurement (the remaining roadmap item)

  # Install Lighthouse CLI if you don't have it
  npm install -g lighthouse

  # Run against local dev
  lighthouse http://localhost:3000 --output html --output-path ./lighthouse-report.html

  # Or just use DevTools > Lighthouse tab — run "Performance" and "Best Practices" audits

  Key metrics to compare before/after: FCP (First Contentful Paint), LCP (Largest Contentful Paint), TTI (Time to Interactive), and the "Serve static assets with an efficient cache policy" audit.
